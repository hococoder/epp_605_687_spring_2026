Overview

In newer iPhones and the Apple Watch Ultra, there is a customizable Action button on the side that can be user configured to carry out custom user actions. Custom user actions? That sounds like a great use of Shortcuts, which were discussed in the last module. It's couldn't be easier to setup the action button with a shortcut you've already defined.

Check your Shortcut

Go ahead and run the example app again, and then open shortcuts and make sure the shortcut we setup last time is still configured with a week.

Setting up the Action Button

Then open up the setting app and choose the Action Button settings. Assign the button to run your selected shortcut.

Firing the Action Button

Then right from the simulator, press and hold down the Action Button (click on it with your mouse and hold). The shortcut will fire just like it did from the shortcuts app. Amazing! So that means any shortcut you create can be triggered from the Action Button.

But what if you want to invoke a shortcut or an intent with your voice. Siri can help with that, and you'll learn about that in the next module.
