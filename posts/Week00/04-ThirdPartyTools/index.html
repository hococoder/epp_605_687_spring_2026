If you have an Apple ID, you already have access to a free developer account. All you have to do is sign into Xcode’s account dialog with those credentials, and Xcode will setup the required certificates you’ll need to develop. If you don’t have an Apple ID already, you can get one at appleid.apple.com.

In this class we’ll be taking it one step further. I have a special Development Team associated with the class, which allows everyone in the class to be part of a development team, allowing you to share projects and resources. This is considered a "paid" developer account, so you also get to access the complete set of capabilities that a developer account can offer! To help with this, please email rsteele3@jhu.edu with your Apple ID and I will add you to the team!

Setting up an account on GitHub

One of your assignments this semester will be the completion of an App Development Log, which will be used to define the scope of your final project later in the semester. We will be using an GitHub's issue tracking to accomplish this. You will need to make an account using your EP address (e.g. @jh.edu). So please do the following:

  • Go to github.com in your browser
  • If you don't already have an account, click on the "Sign up" link in the upper right hand corner
  • Sign up!
  • Register to become part of GitHub Classroom at link here. This is free as part of JHU's GitHub Enterprise program.
  • When you do so, let me know, and I'll add you to the classroom.

Check out the page "Using GitHub Classroom" in Canvas for more details.

Version Control

I highly recommend version control for your projects. Xcode natively supports Git, and will even setup a local git repo for you when creating a project. Since we are using GitHub Classroom, all programming assignments will be under version control. I also suggest (STRONGLY!) doing this for your final project. Note that if I find that your repo is marked Public, I’ll fail you for that assignment. I am more than happy to take a look at code in your repository if you need help – my user email at both sites is hococoder@gmail.com.
