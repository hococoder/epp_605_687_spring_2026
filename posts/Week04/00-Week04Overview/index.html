Welcome to Week 4! This week we'll delve into one of the most important framework for working with mobile apps - Networking!

  • Introduction to Networking: In this module, we'll go over how to communicate with remote systems with URLSession and its family of associated classes, delegates and protocols.
  • Async/Await - a new way to syntactically deal with concurrency, and greatly decreases the amount of networking code you'll write!
  • Syncing with CloudKit: Apple provides a "database in the cloud" experience for storing data - we'll see how easy it is to setup and work with.
  • Introduction to Swift Package Manager: Sometimes it's best not to reinvent the wheel - Swift Package Manager lets developers easily load in external libraries into their project.
  • Programming Assignment 4: In this assignment, you'll build networking functionality into an app so it can communicate with the class server and download assets and other files.
  • Development Log 3: This is it! You've gathered your requirements, sketched some UI, and thought about data flow in your app. In this Development Log, you'll define the final set of tasks in GitHub to include in your final project.
